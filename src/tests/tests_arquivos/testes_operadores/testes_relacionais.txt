inteiro x

    imprima "x é igual a 5"

    imprima "x é diferente de 3"
fim-se

se x > 2 então
    imprima "x é maior que 2"
fim-se

    imprima "x é menor que 10"
fim-se

se x >= 5 então
    imprima "x é maior ou igual a 5"
fim-se

    imprima "x é menor ou igual a 6"
